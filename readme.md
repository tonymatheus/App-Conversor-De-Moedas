# App Conversor de Moedas 

##### O App Tem o objetivo de fazer a  conversão de  valores em dolar para valores em real

**Após Clonar o Repositório em sua máquina execute os seguintes comandos**

>**yarn ou yarn install**

>**yarn start**

>**yarn android**
